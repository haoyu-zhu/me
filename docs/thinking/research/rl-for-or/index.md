# RL for OR

> 用强化学习求解运筹优化问题：当精确求解不可行时，让智能体学出一个快速而可泛化的决策策略。

运筹优化（Operations Research）中存在一大类**组合优化**问题——路径规划、任务调度、资源分配等，其中大多为 NP-hard。随着问题规模增长，基于搜索的算法与传统求解器（solver）的计算开销呈指数级上升；而人工设计的启发式方法依赖领域经验，且往往难以迁移到新的问题结构。

**面向优化的学习（Learning for Optimization, L2O）** 提供了另一条路径：不再人工设计求解规则，而是让神经网络从大量问题实例中学出决策策略。将组合优化问题建模为序贯决策过程（马尔可夫决策过程，MDP）——每一步决定加入哪个节点、执行哪个任务——即可用**强化学习**端到端地训练策略网络。训练完成后，面对新实例只需一次前向推理即可给出高质量解，兼具速度与一定的泛化能力。已有研究表明，此类方法能在保持有竞争力的解质量的同时显著降低计算时间；其核心挑战则在于**泛化**——向更大规模、以及向不同任务结构的迁移。

本研究方向下有两个具体课题，均遵循同一范式：**将问题建模为 MDP → 以注意力网络作为策略 → 用 REINFORCE 端到端训练**，但在问题结构与网络设计上各有侧重。
