# 批量下载财报

> 公司经营分析 · 第 1 步 · 把原始年报批量下下来，A股、港股、美股一站式。

[← 回到「公司经营分析」](index.md)

读财报的第一步是拿到财报。这个 Skill 只做一件事：给它公司名或代码，把对应的年报 / 中报 / 季报 / 招股书原始文件下下来，交给后面的「提取财报经营信息」。它不做任何分析——只负责把最原始、最权威的一手文件搬到本地。

## 设计思路

<div class="skill-idea">核心理念：<b>只搬一手原文，不碰内容</b>——所有数据都从交易所 / 监管机构的官方披露源直接下载，不经二手数据商转手。公司名到代码的解析、市场识别、繁简与格式差异，全部自动处理，用户只管说"下载茅台 2023 年报"。</div>
