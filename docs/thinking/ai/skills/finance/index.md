# 金融

> 把读财报、做估值、看产业链这些重活，沉淀成能反复调用的 Skill。

面向投资研究的一组 Skill，分两条线：一条**向内**，把一家公司的财报读透——抽数据、解读、估值；一条**向外**，顺着一条产业链找出利润捕获在哪、该买哪一环。
