# 产业链分析

> ② 金融 Skill · 顺着一条产业链，找出利润被哪一环长期赚走、该买哪一环。

[← 回到「金融」](index.md)

投资的第一个问题，往往不是"买哪只股"，而是"这条链上，钱到底被哪一环长期赚走"。这个 Skill 站在投研漏斗的最上游：给它一条产业链，它把链条拆开、逐环打分、定位价值捕获点，最后落到一份可投资标的清单。

## 设计思路

<div class="skill-idea">核心理念：<b>先找环节，再找公司</b>——一条产业链上，各环节的赚钱能力天差地别。与其一上来就挑公司，不如先判断"利润在哪一环被稳稳赚走"，再到那一环里找标的。每个判断都靠联网检索的权威报告背书，不依赖本地财报。</div>

分析按投研漏斗自上而下收窄：

<div class="pipe">
  <span class="pipe-step">1 · 拆链<em>环节图谱 / 上下游</em></span>
  <span class="pipe-step">2 · 四问评分<em>需求 / 格局 / 壁垒 / 价值捕获</em></span>
  <span class="pipe-step pipe-key">3 · 短板否决<em>有硬伤一票出局</em></span>
  <span class="pipe-step">4 · 定位价值捕获环节<em>利润堆在哪一层</em></span>
  <span class="pipe-step">5 · 标的清单<em>ticker + 买入纪律 + 风险闸</em></span>
</div>

<div class="disc">
  <span class="disc-item"><b>证据驱动</b>营收 / 毛利 / 份额 / 增速等关键数据靠联网检索权威报告，而非拍脑袋</span>
  <span class="disc-item"><b>找环节，不追热点</b>落点是"哪一环值得投"，而不是"哪个概念最近火"</span>
  <span class="disc-item"><b>短板否决</b>四问里任一项不及格，该环节直接出局，不做加权平均掩盖硬伤</span>
  <span class="disc-item"><b>只出清单，不做组合</b>输出的是待深挖的 ticker 清单，具体买卖与仓位交给下游的单公司深度分析</span>
</div>
