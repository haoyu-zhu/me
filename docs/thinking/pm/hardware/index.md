# 硬件产品经理

> 硬件没有 Ctrl+Z，每一个决定都要为量产负责。

软件做错了可以发版回滚，硬件不行——BOM 定了、模具开了、物料备了，一个判断失误就变成实打实的库存和成本。所以硬件产品经理的功夫，大半花在"做决定之前"：把品类看透、把成本算清、把取舍想明白。

这里放我在硬件方向的思考与实践。目前从**品类深研**开始：一份一份地把市场盘子、产品分层、产业链利润分布和还没被满足的需求拼出来，作为产品定义的地基。后面会陆续补上产品定义、结构与供应链、成本与节奏、软硬件取舍这些方向。

## 已有的报告

**[无人机（2024）](market-research.md)** — 全球 ~260 亿美元、中国 ~325 亿元的盘子，消费级增速骤降而工业级稳步抬升的分化；大疆在消费级 90%+ 的绝对统治，与产品线"零重叠"的价格分层逻辑；并以 DJI Mini 2 拆机为样本做 BOM 拆解，回答整机厂为什么拿走 60%+ 毛利、大疆为什么不自研芯片。

**[消费级手持影像设备（2025）](market-research.md)** — 运动相机与全景相机的品类研究：3–5 百亿级、~2500 万台的全球盘子，全景相机以 20%+ 增速蚕食运动相机；以影石招股书为底还原产业与成本结构（材料占 51%、研发销售双高的"高毛利高研发"模型）；大疆 / 影石 / GoPro / 理光四家的格局；最后落到一个被忽视的潜在需求——卡丁车玩家的"全景 + 遥测 + 复盘"解决方案。
